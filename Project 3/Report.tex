\section{数据集介绍}

本项目使用MSCOCO(Microsoft Common Objects in Context)
数据集的子集，
并使用NOC(novel object captioning)分割方法分割数据集。
数据集中的每张图片都附带5句人工标注的句子。
瓶子、巴士、沙发、微波炉、披萨、球拍、行李箱、斑马等8件物品
不出现在训练集中，但会出现在验证集和测试集中。
